# Zernio DG Carousels

Interactive carousel components for modern web development.

## 🚀 Quick Start

```bash
npm run dev
```

Visit: `http://localhost:3000`

## ✨ Features

- **Smooth Animations** - CSS3 transitions and transforms
- **Auto-play** - 4-second intervals with hover pause
- **Keyboard Navigation** - Arrow key controls
- **Responsive Design** - Works on all screen sizes
- **Modern Styling** - Gradient backgrounds and shadows

## 🛠️ Development

- `npm run dev` - Start development server
- `npm run start` - Alias for dev
- `npm run build` - Build for production

## 📁 Structure

```
├── index.html          # Main page
├── css/style.css       # Styling
├── src/index.js        # Carousel logic
└── package.json        # Configuration
```

## 🎯 Usage

The carousel automatically initializes on page load. Customize slides by editing the HTML structure in `index.html`.

## 🔧 Customization

- **Slides**: Edit slide content in `index.html`
- **Styling**: Modify `css/style.css` for appearance
- **Behavior**: Update `src/index.js` for functionality
